## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas account or local instance
- Mapbox API Key (optional but recommended for maps)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/solo-human.git
   cd solo-human
   ```

2. Setup Frontend:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. Setup Backend:
   ```bash
   cd ../backend
   npm install
   cp .env.example .env # Add your MongoDB URI
   npm start
   ```
